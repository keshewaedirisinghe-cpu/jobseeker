# Changelog

All notable changes to this project will be documented in this file.

## 2026-07-18

### Added

- Added the Milestone 01 project charter defining MVP capabilities, non-goals, user stories, constraints, risks, and success criteria.
- Added ADR 0001 to make human approval mandatory before external communication by default.
- Added ADR 0002 to record the local-first, provider-neutral MVP deployment decision.

### Changed

- Updated Codex workflow rules to allow autonomous full-workflow runs through prompts 00-22.
- Allowed relevant sub-agents for non-overlapping exploration, implementation, review, and test-analysis phases while preserving main-agent accountability.
- Removed remaining user-review/input gates so the agent can continue the complete 00-22 workflow after each milestone passes checks and is committed.
- Updated the README with the approved Milestone 01 architecture summary and operating boundary.
- Updated implementation status to mark Milestone 01 complete and Milestone 02 ready for autonomous continuation.
